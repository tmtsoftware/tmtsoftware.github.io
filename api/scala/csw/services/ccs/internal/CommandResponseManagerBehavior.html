CommandResponseManagerBehavior 

class CommandResponseManagerBehavior extends MutableBehavior[CommandResponseManagerMessage]

The Behavior of a Command Response Manager, represented as a mutable behavior. This behavior will be created as an actor. There will be one CommandResponseManger for a given component which will provide an interface to interact with the status and result of a submitted command
