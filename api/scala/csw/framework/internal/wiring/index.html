  2.  class CswFrameworkSystem extends AnyRef

    A convenient class for creating a typed actor from the provided untyped actor system.

    A convenient class for creating a typed actor from the provided untyped actor system. It creates an actor CswFrameworkGuardian from the provided actor system which spawns any other required actor as its child. This is required because the default supervision strategy for an actor created from untyped actor system is to restart the underlying actor but we want the default supervision strategy of stopping the actor as provided in the typed actor world.

    This actor should be used for spawning actors which needs to be spawned directly from the actor system. This is because the actor system used in csw is an untyped actor system which spawns a typed actor with default supervision strategy of restart rather than stopping the actor which is the default in the typed actor system.
