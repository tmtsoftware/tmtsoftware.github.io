Framework for creating components (HCD, Assembly, Container)

csw-framework library provides support for creating a component as defined by the TMT.

Artifacts

sbt
libraryDependencies += "org.tmt" %% "csw-framework" % "0.1-SNAPSHOT"
maven
<dependency>
 <groupId>org.tmt</groupId>
 <artifactId>csw-framework_2.12</artifactId>
 <version>0.1-SNAPSHOT</version>
 <type>pom</type>
</dependency>
gradle
dependencies {
  compile group: "org.tmt", name: "csw-framework_2.12", version: "0.1-SNAPSHOT"
}
