    Admin Service

    This service provides a handle to admin related APIs which includes logging related APIs

    Admin service has following APIs:

    API Input args Returns
    getLogMetadata componentId LogMetadata
    setLogLevel componentId, level Done

    Creation of Admin Service

    To create Admin client:

    Typescript
    const adminService: AdminService = await AdminService()

    APIs

    getLogMetadata

    This API gives basic logging configuration values for specific component.

    The following example shows how to call getLogMetadata API :

    Typescript
    const prefix = new Prefix('TCS', 'filter.wheel')
const componentId = new ComponentId(prefix, 'HCD')
const logMetaData: LogMetadata = await adminService.getLogMetadata(componentId)

    setLogLevel

    This API allows setting log level of specific component.

    The following example shows how to call setLogLevel API :

    Typescript
    const actionStatus: Done = await adminService.setLogLevel(componentId, 'WARN')
